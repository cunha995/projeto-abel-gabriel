# Painel de Contabilidade Mensal

Este projeto é um painel web para contabilidade mensal dos açudes de peixes e cerâmica/fábrica de tijolos.

## Como rodar

1. Instale as dependências:
   npm install
2. Inicie o projeto:
   npm run dev

## Funcionalidades
- Visualização mensal dos dados
- Painel para açudes de peixes
- Painel para cerâmica/fábrica de tijolos

## Estrutura
- src/App.jsx: Componente principal
- src/main.jsx: Entrada do React
- index.html: HTML base
